# NOMENCLATURA CSS
 
## BEM
 
1. Block (bloque): Componente y/o modulo, independiente UI
2. Element(elemento): Cada elemento que conforma el bloque
3. Modifiter(Modificador): Una variante del bloque que cambia una propiedad del elemento


```
.bloque {...}
.bloque__elemento {...}
.bloque--modificador {...}

```
```
.menu {...}
.menu__item {...}
.menu__item--active {...}
```
